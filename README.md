# рҹҡҖ Installation

Clone the repository

```bash

git clone [https://github.com/RAJK2005/SpaceEscapeRunner.git](https://github.com/RAJK2005/SpaceEscapeRunner.git)

```

Open the project

```bash

cd SpaceEscapeRunner

```

Install dependencies

```bash

npm install

```

Run the project

```bash

npx expo start

```

Run on Android

```bash

npx expo run:android

```
